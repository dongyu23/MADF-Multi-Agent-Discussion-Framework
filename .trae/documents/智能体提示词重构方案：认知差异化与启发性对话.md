# 智能体“启发性”思维重构方案

针对当前智能体存在的“思维同质化”和“性格脸谱化”问题，本方案将对 Prompt 进行第三次深度迭代。核心目标是将智能体从“扮演角色的演员”转变为**“持有特定思维模型的思想者”**。

## 1. 认知差异化重构 (`god.py`)

**目标**：废除通用的 MBTI 和性格缺陷设定，改为分配**具体的思维模型 (Mental Models)**。

*   **Prompt 修改**:
    *   **移除**：`mbti`, `flaw`, `style`（性格/语言风格）。
    *   **新增**：`mental_model`（核心思维模型），如：第一性原理、博弈论、演化心理学、公地悲剧、熵增定律等。
    *   **新增**：`focus_perspective`（关注视角），如：物理极限、资源分配、历史重演、系统稳定性。
    *   **生成逻辑**：上帝系统不再生成“性格各异的人”，而是生成“持有不同认知工具的专家”。

## 2. 思考逻辑重构 (`agent.py` - `think`)

**目标**：打破“升维/降维/联想”的通用模板，强制使用分配的思维模型进行**深度解构**。

*   **Prompt 修改**:
    *   **移除**：`processing: {up, down, link}` 的固定结构。
    *   **新增**：`analysis`（模型分析），要求智能体**显式调用**自己的 `mental_model` 来拆解当前话题。
    *   **指令**：
        *   “不要使用通用的逻辑（如利弊分析）。”
        *   “必须严格基于你的【核心思维模型】（如熵增定律）进行推演。”
        *   “寻找系统中的**变量、反馈回路、临界点**，而不是谈论观点。”

## 3. 发言机制重构 (`agent.py` - `speak`)

**目标**：建立“信息增量”标准和“建设性对抗”机制。

*   **Prompt 修改**:
    *   **引入“钢铁侠重塑” (Steelmanning)**：在反驳前，必须先承认对方观点中合理的那 10%（前提是符合逻辑），然后指出其适用边界或二阶后果。
    *   **信息增量奖惩**：
        *   **禁止**：重复观点、和稀泥（“既要又要”）、情绪化攻击。
        *   **必须**：引用具体概念/模型、指出逻辑缺失环、提供独特视角。
    *   **输出要求**：多使用类比，少使用形容词；让听众产生“原来还可以从这个角度看问题”的顿悟感。

## 4. 实施计划

1.  **Refactor God**: 修改 `generate_personas`，不再生成性格，而是生成思维模型。
2.  **Refactor Think**: 修改 `think` 函数，移除旧模板，植入基于思维模型的分析指令。
3.  **Refactor Speak**: 修改 `speak` 函数，植入 Steelmanning 和信息增量约束。
4.  **Verify**: 运行“Universal Basic Income (UBI)”或“Climate Change”话题，验证是否产生高质量的思想碰撞而非吵架。
